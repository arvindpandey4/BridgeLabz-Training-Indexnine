# Node.js and JavaScript Practice

Complete Node.js and JavaScript fundamentals practice files.

## Files

01_fundamentals.js - Type conversions, scope, closures, recursion, spread syntax
02_functions_advanced.js - IIFE, NFE, call/apply/bind, new Function, scheduling, decorators
03_event_loop_oop.js - Event loop, event handlers, OOP, classes, inheritance
04_practice_examples.js - Closures, memoization, currying, emitter, promises, async/await, proxy

## Installation

npm install

## Run

npm start
node 01_fundamentals.js
node 02_functions_advanced.js
node 03_event_loop_oop.js
node 04_practice_examples.js

## Topics

NodeJS, NPM, OOP, Event Loop, Event handlers, Type Conversions, use strict, Comparisons, Conditional branching, Logical & Nullish coalescing operator, Polyfills and transpilers, Recursion and stack, Rest parameters and spread syntax, Variable scope, closure, IIFE, NFE, The "new Function" syntax, Scheduling, Decorators and forwarding, call/apply, Function binding
