# BridgeLabz Training - TypeScript

This branch contains TypeScript implementation of various concepts and the Address Book problem.

## Files
* `01_types_interfaces.ts`: Basic types and interface definitions.
* `02_classes_oop.ts`: Object-Oriented Programming with Classes.
* `03_access_modifiers.ts`: Public, private, and protected modifiers.
* `04_decorators.ts`: Implementation of decorators.
* `05_exception_handling.ts`: Error handling and custom exceptions.
* `06_logging.ts`: Logging implementation.
* `07_file_io.ts`: Reading and writing files.
* `08_address_book.ts`: Address Book project.
* `09_generics.ts`: Generic types and functions.
* `10_coding_standards.ts`: Code formatting and standards.
* `11_advanced_types.ts`: Advanced type definitions.
* `12_data_structures.ts`: Data structure implementations.
* `practice.ts`: Basic arithmetic operations and error handling practice.
