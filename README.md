# Advanced JavaScript & OOP Practice

This branch contains advanced JavaScript concepts and Object-Oriented Programming (OOP) implementations.

## Topics Covered

### Core OOP Concepts
- **classes_inheritance_extending_builtin.js** - ES6 Classes, Inheritance, and Built-in Object Extension
- **constructor_new_this.js** - Constructor functions, `new` keyword, and `this` binding
- **prototypes_inheritance_native.js** - Prototype chain and native inheritance
- **objects_references_copying.js** - Object references, shallow/deep copying

### Object Features
- **object_methods_optional_chaining.js** - Object methods and optional chaining (?.)
- **object_to_primitive_descriptors.js** - Object-to-primitive conversion and property descriptors

### Asynchronous Programming
- **callbacks_promises_chaining.js** - Callbacks, Promises, and Promise chaining
- **async_await_promisify_microtasks.js** - Async/await, Promisify, and Microtasks
- **promise_api_custom_error.js** - Promise API and custom error handling

### Advanced JavaScript
- **generators_async_iterators_currying_hoisting.js** - Generators, Async Iterators, Currying, and Hoisting

## Learning Outcomes

- Master JavaScript OOP patterns (Constructor, Prototype, Class-based)
- Understand `this` binding and inheritance models
- Work with Promises and Async/await effectively
- Handle advanced concepts like generators and currying
- Build scalable and maintainable JavaScript applications
