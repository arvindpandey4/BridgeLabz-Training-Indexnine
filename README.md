# Data Structures & Algorithms (DSA) Practice

This directory contains JavaScript implementations of fundamental data structures and algorithms.

## Included Topics

### Searching Algorithms
- **linear_search.js** - Linear/Sequential Search
- **binary_search.js** - Binary Search

### Sorting Algorithms
- **bubble_sort.js** - Bubble Sort
- **selection_sort.js** - Selection Sort
- **insertion_sort.js** - Insertion Sort
- **merge_sort.js** - Merge Sort
- **quick_sort.js** - Quick Sort

### Data Structures
- **array_basics.js** - Array operations and basics
- **linked_list.js** - Linked List implementation

## Usage

Each file contains practice code demonstrating the implementation of the respective algorithm or data structure.

## Learning Outcomes

- Understand core searching and sorting techniques
- Learn Big O time and space complexity
- Practice implementing common data structures
- Build problem-solving skills
