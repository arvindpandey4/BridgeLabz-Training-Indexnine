# BridgeLabz Training

## Practice Files
This section contains various JavaScript scripts used to practice and demonstrate core language concepts. It includes examples for:
- Array methods
- Callbacks and Promises
- Closures and Currying
- Hoisting and Temporal Dead Zone (TDZ)
- Keyed Collections (Maps and Sets)
- Spread and Rest operators
- String and Number manipulation

## Employee Wage Project
This project implements an Employee Wage Calculation system using JavaScript. It follows a progressive development approach with multiple use cases:
1. Employee attendance checking
2. Daily wage calculation
3. Handling part-time and full-time logic using switch cases
4. Calculating monthly wages with working hour limits
5. Refactoring to use Class and Object-Oriented principles
6. Supporting wage calculations for multiple companies
